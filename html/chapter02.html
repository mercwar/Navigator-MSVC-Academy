Chapter 2 — Your First C Program

Element Description
Source Code 

#include <stdio.h>

int main(void) {
    printf("Hello, C world!\\n");
    return 0;
}
This is the classic “Hello, world!” program. It demonstrates the basic structure of a C program.
#include <stdio.h> Imports the Standard I/O library. Provides functions like printf, scanf, puts, and more.
main() function int main(void) is the entry point of every C program. The operating system calls this function when the program starts.
  • int — return type
  • main — required function name
  • void — no parameters
printf() Prints formatted text to the console. \\n adds a newline. 

printf("Hello, C world!\\n");
return 0; Returns 0 to the operating system, signaling successful execution. Non‑zero values indicate errors.
Compiling (Linux/macOS) 

gcc main.c -o main
./main
  • gcc — GNU C Compiler
  • -o main — output executable named main
  • ./main — run the program
Compiling (Windows MSVC) 

cl main.c
main.exe
  • cl — Microsoft Visual C compiler
  • main.exe — generated executable
Common beginner mistakes
  • Missing semicolons
  • Incorrect quotes
  • Forgetting #include <stdio.h>
  • Using main() instead of int main(void)
