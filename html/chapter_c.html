Chapter C — Notepad++ Installation

StepRoutine
Download 

Go to: https://notepad-plus-plus.org/downloads/
Download latest 64-bit installer (e.g., npp.8.6.4.Installer.x64.exe)
Install 

Run installer → Accept license
Install to: C:\Program Files\Notepad++
Select components:
- Core files
- Plugins Admin
- Auto-completion files
Verify 

Open Notepad++
Check version under Help → About Notepad++
Configure 

Settings → Preferences:
- Enable Dark Mode
- Set default encoding: UTF-8
- Enable Auto-completion
Plugins 

Plugins → Plugins Admin → Install:
- NppFTP (FTP/SFTP integration)
- Compare (file diff)
- JSON Viewer
- XML Tools
- Explorer sidebar
Notes 

Navigator‑MSVC Academy uses Notepad++ for:
- Quick edits of batch files, configs, and scripts
- Lightweight IDE integration
- Plugin-based file transfer and comparison
