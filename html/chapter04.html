Chapter 4 — Variables & Data Types

Concept Description
What is a variable? A variable is a named storage location in memory. It holds data that your program can read or modify.

In C, every variable has:
  • a type
  • a name
  • a value
Integer (int) Stores whole numbers. 

int age = 25;
Range depends on system (typically −2,147,483,648 to 2,147,483,647).
Floating point (float) Stores decimal numbers with single precision. 

float height = 1.82f;
Requires f suffix for literal floats.
Double precision (double) Stores decimal numbers with double precision. 

double pi = 3.1415926535;
More accurate than float.
Character (char) Stores a single character. 

char initial = 'A';
Uses single quotes.
Multiple declarations You can declare several variables at once: 

int x = 10, y = 20, z = 30;
Uninitialized variables C does not automatically initialize variables. 

int value;  // contains garbage data
Always initialize your variables.
Constants Use const to prevent modification: 

const int maxScore = 100;
Attempting to change maxScore will cause a compiler error.
Type sizes Use sizeof to check memory size: 

printf("%zu\\n", sizeof(int));
printf("%zu\\n", sizeof(double));
Typical sizes:
  • int — 4 bytes
  • float — 4 bytes
  • double — 8 bytes
  • char — 1 byte
Type modifiers C supports modifiers:
  • short
  • long
  • unsigned
Example: 

unsigned int score = 500;
unsigned removes negative values.
Scope Variables exist only within their scope:
  • local — inside functions
  • global — outside functions
Example: 

int globalValue = 10;

int main(void) {
    int localValue = 5;
}
Shadowing A local variable can hide a global variable: 

int value = 10;

int main(void) {
    int value = 20;  // shadows global
}
