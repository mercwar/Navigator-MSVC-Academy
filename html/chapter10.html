Chapter 10 — Pointers in C

Concept Description
What is a pointer? A pointer is a variable that stores a memory address. Instead of holding a value directly, it points to where the value is stored.

Example: 

int value = 10;
int *ptr = &value;
  • value — stores 10
  • &value — address of value
  • ptr — stores that address
Dereferencing Dereferencing means accessing the value stored at the pointer’s address. 

printf("%d\\n", *ptr);  // prints 10
*ptr gives the value, not the address.
Pointer declaration 

int *p;
float *f;
char *c;
The * indicates the variable is a pointer.
Pointer assignment 

int x = 50;
int *p = &x;
Pointers must be assigned valid addresses.
NULL pointer A pointer that points to nothing: 

int *p = NULL;
Always check pointers before dereferencing.
Pointer arithmetic Adding to a pointer moves it by the size of its type: 

int nums[3] = {10, 20, 30};
int *p = nums;

printf("%d\\n", *(p + 1));  // prints 20
p + 1 moves one integer forward.
Arrays and pointers Arrays decay to pointers: 

int nums[3] = {1, 2, 3};
int *p = nums;  // same as &nums[0]
Access: 

printf("%d\\n", p[2]);  // prints 3
Pointers to pointers 

int x = 10;
int *p = &x;
int **pp = &p;
  • p points to x
  • pp points to p
Pointer parameters Passing pointers allows functions to modify variables: 

void set(int *p) {
    *p = 100;
}
Usage: 

int x = 5;
set(&x);  // x becomes 100
Dynamic memory Pointers are required for dynamic memory: 

int *p = malloc(sizeof(int) * 5);
Always free memory: 

free(p);
Void pointers 

void *ptr;
Can point to any type, but must be cast before dereferencing.
Function pointers 

int add(int a, int b) { return a + b; }

int (*fp)(int, int) = add;
printf("%d\\n", fp(3, 4));  // prints 7
Used in callbacks and advanced C programming.
Common pointer bugs
  • Dereferencing NULL
  • Using uninitialized pointers
  • Memory leaks (forgetting free)
  • Double free
  • Out-of-bounds pointer arithmetic
Example: Full pointer program 

#include <stdio.h>

void update(int *p) {
    *p = 42;
}

int main(void) {
    int value = 10;
    int *ptr = &value;

    printf("Before: %d\\n", value);
    update(ptr);
    printf("After: %d\\n", value);

    return 0;
}
