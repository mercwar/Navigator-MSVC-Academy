Chapter 5 — Input & Output in C

Concept Description
Standard I/O Library All input and output in C is handled through the Standard I/O library: 
#include <stdio.h>
It provides:
  • printf — formatted output
  • scanf — formatted input
  • puts — simple string output
  • gets (unsafe) — deprecated input
printf() Prints formatted text to the console. 

printf("Age: %d\\n", age);
printf("Height: %.2f\\n", height);
Format specifiers:
  • %d — integer
  • %f — float/double
  • %c — character
  • %s — string
scanf() Reads formatted input from the user. 

int age;
printf("Enter age: ");
scanf("%d", &age);
Important:
  • scanf requires the & operator to store input into variables.
  • Always validate input to avoid errors.
Reading multiple values 

int a, b;
scanf("%d %d", &a, &b);
The user enters two numbers separated by spaces.
Reading floats 

float temp;
scanf("%f", &temp);
Reading characters 

char letter;
scanf(" %c", &letter);
Note the leading space — it prevents newline issues.
Reading strings 

char name[50];
scanf("%49s", name);
Never use gets() — it is unsafe.
puts() Simple string output: 

puts("Hello!");
Automatically adds a newline.
putchar() Prints a single character: 

putchar('A');
Escape sequences
  • \\n — newline
  • \\t — tab
  • \\\\ — backslash
  • \\" — quote
Example: 

printf("Line1\\nLine2\\n");
Input pitfalls
  • scanf leaves newline characters in the buffer
  • Reading char often requires " %c"
  • Strings must have enough space
  • Invalid input can break the program
Example: Full I/O Program 

#include <stdio.h>

int main(void) {
    int age;
    float height;
    char initial;

    printf("Enter age: ");
    scanf("%d", &age);

    printf("Enter height: ");
    scanf("%f", &height);

    printf("Enter initial: ");
    scanf(" %c", &initial);

    printf("You are %d years old, %.2f tall, initial %c.\\n",
           age, height, initial);

    return 0;
}
