Chapter H — Git + GitHub CLI Installation

StepRoutine
Download Git 

Go to: https://git-scm.com/download/win
Download latest Git for Windows installer
Install Git 

Run installer → Accept license
Options:
- Select editor (Notepad++ or VS Code)
- Add Git to PATH
- Enable Git Credential Manager
Verify Git 

Open CMD
Run: git --version
Output should display Git version info
Install GitHub CLI 

Option 1: winget install GitHub.cli
Option 2: Download MSI from https://github.com/cli/cli/releases
Authenticate GitHub CLI 

Run: gh auth login
Choose:
- GitHub.com
- HTTPS
- Paste personal access token or authenticate via browser
Verify GitHub CLI 

Run: gh --version
Run: gh repo list
Output should display repositories
Notes 

Navigator‑MSVC Academy uses Git + GitHub CLI for:
- Version control of FireGem builds
- Automated repo management
- Command fusion with LLM execution
