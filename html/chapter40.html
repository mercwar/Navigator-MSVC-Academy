Chapter 40 — Asset Pipeline: Importing, Converting, Compressing & Optimizing

Topic Description
Asset Pipeline Overview A modern game engine uses a **content pipeline** to process raw assets into optimized runtime formats. Pipeline stages:
  • Import
  • Convert
  • Optimize
  • Compress
  • Package
  • Load at runtime
Raw assets → processed assets → runtime assets.
Importers Importers read external formats:
  • FBX
  • OBJ
  • GLTF
  • PNG/JPG/TGA
  • WAV/OGG
Example importer interface: 

struct ImportResult {
    Mesh mesh;
    Material material;
    Animation anim;
};

class Importer {
public:
    virtual ImportResult Import(const char *path) = 0;
};
Mesh Conversion Convert imported mesh into engine format: 

struct Vertex {
    XMFLOAT3 pos;
    XMFLOAT3 normal;
    XMFLOAT2 uv;
    XMFLOAT4 tangent;
};

struct Mesh {
    vector verts;
    vector indices;
};
Generate tangents: 

ComputeTangents(mesh.verts, mesh.indices);
Normalize normals: 

NormalizeNormals(mesh.verts);
Mesh Optimization Optimize for GPU vertex cache: 

OptimizeVertexCache(mesh.indices);
OptimizeOverdraw(mesh.indices, mesh.verts);
OptimizeVertexFetch(mesh.indices, mesh.verts);
Tools like **meshoptimizer** perform:
  • Vertex cache optimization
  • Overdraw reduction
  • Vertex fetch optimization
Texture Processing Convert textures to GPU formats:
  • BC1 (DXT1)
  • BC3 (DXT5)
  • BC7 (high quality)
  • BC5 (normal maps)
Example: 

CompressBC7("albedo.png", "albedo.bc7");
CompressBC5("normal.png", "normal.bc5");
Generate mipmaps: 

GenerateMipmaps(texture);
Normal Map Conversion Convert RGB normal map → BC5 two‑channel: 

for each pixel:
    bc5.r = normal.x * 0.5 + 0.5;
    bc5.g = normal.y * 0.5 + 0.5;
Z reconstructed in shader: 

float z = sqrt(1 - x*x - y*y);
Animation Baking Bake skeletal animation into engine format: 

struct Keyframe {
    float time;
    XMFLOAT3 pos;
    XMFLOAT4 rot;
    XMFLOAT3 scale;
};

struct Animation {
    vector> bones;
};
Compress keyframes: 

QuantizeRotations(anim);
RemoveRedundantKeys(anim);
Skeleton Processing Skeleton structure: 

struct Bone {
    int parent;
    XMMATRIX bindPose;
};
Precompute inverse bind pose: 

bone.invBind = XMMatrixInverse(nullptr, bone.bindPose);
Audio Processing Convert WAV → OGG: 

ConvertToOgg("sound.wav", "sound.ogg");
Generate streaming chunks: 

SplitAudioIntoChunks("music.ogg");
Precompute FFT for visualization: 

ComputeFFT(audioData);
Asset Packaging Package assets into bundles: 

struct Bundle {
    vector headers;
    vector data;
};
Add asset: 

AddToBundle(bundle, "mesh", meshData);
AddToBundle(bundle, "texture", texData);
Compress bundle: 

CompressLZ4(bundle.data);
Runtime Loading Load bundle: 

Bundle b = LoadBundle("level.bundle");
Lookup asset: 

AssetHeader *h = FindAsset(b, "player.mesh");
Mesh m = LoadMesh(b.data + h->offset);
Hot Reload Watch asset directory: 

if (FileChanged(path))
    ReloadAsset(path);
Allows editing assets while game runs.
Full Asset Pipeline Summary 

// 1. Import raw assets
ImportFBX();
ImportTextures();
ImportAudio();

// 2. Convert to engine formats
ConvertMeshes();
ConvertTextures();
ConvertAnimations();

// 3. Optimize
OptimizeMeshes();
CompressTextures();
BakeAnimations();

// 4. Package
BuildBundles();

// 5. Runtime
LoadBundles();
StreamAssets();
HotReload();
