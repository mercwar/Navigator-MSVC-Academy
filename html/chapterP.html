Chapter P — AVIS Arbitration Layer: Multi‑Subsystem DX Command Control

TopicDescription
Arbitration Concept AVIS decides **which subsystem** may issue DirectX commands at any moment. 

enum AVIS_OWNER {
    OWNER_CBORD,
    OWNER_CYBORG,
    OWNER_MERCG,
    OWNER_NONE
};
DX Ownership Token 

struct AVIS_TOKEN {
    AVIS_OWNER owner;
    uint64_t   frameID;
};
Only the owner may vector into DirectX for that frame.
Token Assignment 

AVIS_TOKEN AVIS_AssignToken()
{
    if (avis.cyhy->dxHealth.deviceLost)
        return { OWNER_CBORD, avis.frameID };

    return { OWNER_MERCG, avis.frameID };
}
