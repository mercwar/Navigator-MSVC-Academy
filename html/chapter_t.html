MySQL / MariaDB Integration 

Download MariaDB: https://mariadb.org/download/
Install to: C:\MariaDB
Configure my.ini:
[mysqld]
port=3306
datadir=C:/MariaDB/data
sql_mode=STRICT_TRANS_TABLES

Test:
mysql -u root -p
PHP Database Extension 

Edit php.ini:
extension=mysqli
extension=pdo_mysql

Restart Apache
Verify with:
SSL Certificates 

Generate self-signed cert:
openssl req -new -x509 -days 365 -nodes -out server.crt -keyout server.key

Edit httpd-ssl.conf:
SSLCertificateFile "C:/Apache24/conf/server.crt"
SSLCertificateKeyFile "C:/Apache24/conf/server.key"

Enable mod_ssl
Restart Apache
FireGem Engine Hooks 

Create hooks directory: C:\Projects\Navigator-MSCV\hooks
Example hook: build_hook.bat
@echo off
echo Running FireGem build hook...
cl /O2 main.cpp -Fe:firegem.exe
GitHub Actions Integration 

Create .github/workflows/build.yml:
name: FireGem Build
on: [push]
jobs:
  build:
    runs-on: windows-latest
    steps:
    - uses: actions/checkout@v3
    - name: Build
      run: cl /O2 main.cpp -Fe:firegem.exe
