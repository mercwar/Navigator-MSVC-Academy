Chapter 6 — Conditionals in C

Concept Description
What are conditionals? Conditionals allow your program to make decisions. They evaluate expressions and execute different code depending on whether the expression is true or false.

C uses:
  • if
  • else
  • else if
  • switch
Basic if statement 

int age = 20;

if (age >= 18) {
    printf("Adult\\n");
}
The code inside the if block runs only if the condition is true.
if / else 

if (age >= 18) {
    printf("Adult\\n");
} else {
    printf("Minor\\n");
}
else runs when the if condition is false.
else if chain 

int score = 85;

if (score >= 90) {
    printf("A\\n");
} else if (score >= 80) {
    printf("B\\n");
} else if (score >= 70) {
    printf("C\\n");
} else {
    printf("D or F\\n");
}
Evaluates conditions in order until one is true.
Comparison operators
  • == equal to
  • != not equal to
  • > greater than
  • < less than
  • >= greater or equal
  • <= less or equal
Example: 

if (x != 10) {
    printf("x is not 10\\n");
}
Logical operators
  • && — logical AND
  • || — logical OR
  • ! — logical NOT
Example: 

if (age >= 18 && age <= 65) {
    printf("Working age\\n");
}
Boolean expressions In C, true is any non-zero value. false is zero. 

if (1) { printf("true\\n"); }
if (0) { printf("false\\n"); }
Switch statement 

int option = 2;

switch (option) {
    case 1:
        printf("Option 1\\n");
        break;
    case 2:
        printf("Option 2\\n");
        break;
    case 3:
        printf("Option 3\\n");
        break;
    default:
        printf("Unknown option\\n");
}
  • switch compares one value
  • case defines possible matches
  • break prevents fall-through
  • default handles unmatched values
Fall-through behavior Without break, execution continues into the next case: 

switch (x) {
    case 1:
        printf("One\\n");
    case 2:
        printf("Two\\n");
}
If x == 1, both lines print.
Ternary operator A compact conditional expression: 

int age = 20;
const char *type = (age >= 18) ? "Adult" : "Minor";
Equivalent to an if / else.
Common mistakes
  • Using = instead of ==
  • Forgetting break in switch
  • Misusing logical operators
  • Missing braces in nested conditionals
Example: Full conditional program 

#include <stdio.h>

int main(void) {
    int temp;

    printf("Enter temperature: ");
    scanf("%d", &temp);

    if (temp > 30) {
        printf("Hot\\n");
    } else if (temp > 20) {
        printf("Warm\\n");
    } else if (temp > 10) {
        printf("Cool\\n");
    } else {
        printf("Cold\\n");
    }

    return 0;
}
