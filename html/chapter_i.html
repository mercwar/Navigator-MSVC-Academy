Chapter I — Accounts Setup (Google + GitHub)

StepRoutine
Create Google Account 

Go to: https://accounts.google.com
Click: Create account → For myself
Enter:
- First & Last Name
- Birthday & Gender
Choose username & password
Add recovery phone/email
Verify via SMS or email
Configure Google Account 

Enable 2-Step Verification
Set recovery options
Sync with Chrome for bookmarks & passwords
Create GitHub Account 

Go to: https://github.com
Click: Sign up
Enter:
- Username
- Email
- Password
Verify email
Configure GitHub Account 

Enable 2FA (SMS or Authenticator app)
Set profile details
Create first repository:
- Public or Private
- Initialize with README
Integrate with GitHub CLI 

Run: gh auth login
Select:
- GitHub.com
- HTTPS
Authenticate via browser or token
Verify: gh repo list
Notes 

Navigator‑MSVC Academy requires:
- Google account for cloud sync & authentication
- GitHub account for repo management & FireGem builds
