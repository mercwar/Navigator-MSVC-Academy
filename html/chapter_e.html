Chapter E — PHP Installation

StepRoutine
Download 

Go to: https://windows.php.net/download/
Download latest Thread Safe ZIP (e.g., php-8.3.9-Win32-vs17-x64.zip)
Install 

Extract ZIP to: C:\php
Rename php.ini-development → php.ini
Configure 

Edit php.ini:
- Enable extensions: extension_dir = "ext"
- Uncomment: extension=mysqli, extension=curl, extension=openssl
Integrate with Apache 

Edit C:\Apache24\conf\httpd.conf:
LoadModule php_module "C:/php/php8apache2_4.dll"
AddHandler application/x-httpd-php .php
PHPIniDir "C:/php"
Verify 

Create file: C:\Apache24\htdocs\info.php
Content:

Open browser → http://localhost/info.php
Output should display PHP info page
Notes 

Navigator‑MSVC Academy uses PHP for:
- Dynamic web content
- GF engine integration
- LLM command execution endpoints
