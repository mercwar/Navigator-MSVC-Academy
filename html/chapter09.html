Chapter 9 — Arrays in C

Concept Description
What is an array? An array is a sequence of elements stored in contiguous memory. All elements share the same type and are accessed by index.

Example: 

int nums[5];
This creates space for 5 integers.
Array initialization 

int nums[5] = {1, 2, 3, 4, 5};
You can also let the compiler count: 

int nums[] = {10, 20, 30};
Accessing elements 

printf("%d\\n", nums[0]);  // first element
printf("%d\\n", nums[4]);  // fifth element
Indexing starts at 0.
Modifying elements 

nums[2] = 99;
Changes the third element.
Looping through arrays 

for (int i = 0; i < 5; i++) {
    printf("%d\\n", nums[i]);
}
Arrays and loops are often used together.
Out-of-bounds access 

nums[10] = 123;  // ERROR: undefined behavior
C does not check bounds — accessing invalid indices can corrupt memory.
Character arrays (strings) 

char name[6] = "Demon";
Includes a hidden \\0 terminator.

Equivalent: 

char name[] = {'D','e','m','o','n','\\0'};
Multidimensional arrays 

int grid[3][3] = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};
Access: 

printf("%d\\n", grid[1][2]);  // prints 6
Memory layout Arrays are stored in contiguous memory: 

int nums[3] = {10, 20, 30};
  • nums[0] at address X
  • nums[1] at X + sizeof(int)
  • nums[2] at X + 2 * sizeof(int)
Arrays and pointers Arrays decay to pointers: 

int nums[3] = {1, 2, 3};
int *p = nums;  // same as &nums[0]
Access: 

printf("%d\\n", *(p + 1));  // prints 2
Passing arrays to functions 

void show(int arr[], int size) {
    for (int i = 0; i < size; i++)
        printf("%d\\n", arr[i]);
}
Arrays are passed as pointers.
Returning arrays C cannot return arrays directly. Use pointers or structs instead. 

int* makeArray(void) {
    static int nums[3] = {1, 2, 3};
    return nums;
}
static keeps the array alive after the function ends.
Example: Full array program 

#include <stdio.h>

int main(void) {
    int nums[5] = {5, 10, 15, 20, 25};

    for (int i = 0; i < 5; i++) {
        printf("nums[%d] = %d\\n", i, nums[i]);
    }

    return 0;
}
