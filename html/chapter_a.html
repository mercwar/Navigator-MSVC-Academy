Chapter A — MSVC Build Tools Installation

StepRoutine
Download 

Go to: https://visualstudio.microsoft.com/visual-cpp-build-tools/
Download the latest Visual Studio Build Tools installer
Install 

Run installer → Select workloads:
- MSVC v143 Build Tools
- Windows 10/11 SDK
- C++ CMake tools for Windows
Verify 

Open "x64 Native Tools Command Prompt for VS 2022"
Run: cl
Output should show MSVC version info
Notes 

Navigator‑MSVC Academy uses MSVC for:
- GF-only builds
- LLM command execution modules
CMD only — no PowerShell
