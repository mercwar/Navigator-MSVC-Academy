Chapter N — Troubleshooting

IssueRoutine
MSVC Compiler Not Found 

Symptom: 'cl' not recognized
Fix:
- Open "x64 Native Tools Command Prompt"
- Ensure PATH includes MSVC bin
- Reinstall Build Tools workload
MSYS2 Update Loop 

Symptom: pacman repeatedly asks to update
Fix:
- Run: pacman -Syu
- Close shell → reopen
- Run: pacman -Su
Apache Fails to Start 

Symptom: httpd.exe exits immediately
Fix:
- Run: httpd -t → check syntax
- Verify Listen port not in use
- Check logs: C:\Apache24\logs\error.log
PHP Not Loading 

Symptom: .php files download instead of execute
Fix:
- Ensure php_module DLL path correct in httpd.conf
- Verify PHPIniDir points to C:\php
- Restart Apache
WinSCP Connection Error 

Symptom: Cannot connect to server
Fix:
- Verify hostname/IP
- Check firewall/port (22 for SFTP, 21 for FTP)
- Test with PuTTY or ssh
7-Zip Command Not Found 

Symptom: '7z' not recognized
Fix:
- Add C:\Program Files\7-Zip to PATH
- Verify installation directory
GitHub CLI Auth Failure 

Symptom: gh auth login fails
Fix:
- Use browser authentication
- Ensure token has repo scope
- Run: gh auth status
Notes 

Navigator‑MSVC Academy troubleshooting relies on:
- CMD logs
- Apache error.log
- PHP info.php
- GitHub CLI status
