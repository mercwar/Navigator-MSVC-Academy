Chapter U — Academy Workflow

StageRoutine
Build Stage 

Run BUILD.bat
- Compiles source with MSVC
- Outputs app.exe to build directory
Logs stored in build.log
Package Stage 

Run PACKAGE.bat
- Compresses build directory into release.zip
- Stores package in release folder
Deploy Stage 

Run DEPLOY.bat
- Syncs local project with remote server via WinSCP
- Updates Apache htdocs
Validation Stage 

Open browser → http://localhost/test.php
Output: Navigator-MSCV OK
Check remote server for deployed files
Monitoring Stage 

Daily:
- Review Apache error.log
- Review php_errors.log
- Check build.log for warnings
Weekly:
- Run systeminfo + tasklist
Automation Stage 

Schedule tasks with Task Scheduler:
- BUILD.bat daily at 9 AM
- PACKAGE.bat weekly Sunday
- DEPLOY.bat after successful build
Notes 

Navigator‑MSVC Academy workflow ensures:
- Builds compiled
- Packages created
- Deployments synchronized
- Validation confirmed
- Monitoring continuous
- Automation scheduled
CMD only — no PowerShell
