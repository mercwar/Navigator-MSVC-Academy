Chapter F — WinSCP Installation

StepRoutine
Download 

Go to: https://winscp.net
Download latest WinSCP installer (e.g., WinSCP-6.3.2-Setup.exe)
Install 

Run installer → Accept license
Choose installation type: Typical Installation
Select interface style: Commander (dual-pane) or Explorer
Verify 

Launch WinSCP
Check version under Help → About
Configure 

Create new site connection:
- Protocol: SFTP / FTP / SCP
- Hostname: server address
- Port: 22 (SFTP/SSH), 21 (FTP)
- Username + Password or Key
Save session for reuse
Integration 

Enable drag-and-drop file transfers
Set default local directory: C:\Projects\Navigator-MSCV
Set default remote directory: /var/www/html
Notes 

Navigator‑MSVC Academy uses WinSCP for:
- Secure file transfers to Apache/PHP servers
- Synchronizing project directories
- Quick deployment of FireGem builds
