Chapter 7 — Loops in C

Concept Description
What are loops? Loops allow your program to repeat actions automatically. C provides three main loop types:
  • for — counted loops
  • while — condition-based loops
  • do-while — guaranteed one execution
for loop 

for (int i = 0; i < 5; i++) {
    printf("i = %d\\n", i);
}
  • i = 0 — initialization
  • i < 5 — condition
  • i++ — increment
Runs 5 times (0–4).
while loop 

int count = 0;

while (count < 3) {
    printf("count = %d\\n", count);
    count++;
}
Repeats while the condition is true.
do-while loop 

int x = 0;

do {
    printf("x = %d\\n", x);
    x++;
} while (x < 3);
Always runs at least once.
Infinite loops 

while (1) {
    printf("Running forever\\n");
}
Use carefully — often used in embedded systems and game loops.
break statement 

for (int i = 0; i < 10; i++) {
    if (i == 5) break;
}
break exits the loop immediately.
continue statement 

for (int i = 0; i < 10; i++) {
    if (i % 2 == 0) continue;
    printf("%d\\n", i);
}
continue skips the current iteration.
Nested loops 

for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        printf("%d, %d\\n", i, j);
    }
}
Useful for grids, matrices, and multi-dimensional data.
Loop pitfalls
  • Forgetting to update the loop variable
  • Infinite loops caused by bad conditions
  • Off-by-one errors
  • Using == instead of < or >
Example: Full loop program 

#include <stdio.h>

int main(void) {
    for (int i = 1; i <= 10; i++) {
        if (i % 2 == 0) {
            printf("%d is even\\n", i);
        } else {
            printf("%d is odd\\n", i);
        }
    }
    return 0;
}
