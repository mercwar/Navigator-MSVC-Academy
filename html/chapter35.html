Chapter 35 — Screen‑Space Effects: SSAO, SSR, Fog & Advanced GPU Post‑Processing

Topic Description
What Are Screen‑Space Effects? Screen‑space effects operate **after** geometry is rendered, using:
  • Depth buffer
  • Normal buffer
  • Color buffer
They allow:
  • Ambient occlusion
  • Reflections
  • Fog
  • Edge detection
  • Depth‑based blur
All effects use **full‑screen passes**.
G‑Buffer Setup For SSAO/SSR you need:
  • Depth texture
  • Normal texture
  • Color texture
Normal buffer creation: 

D3D11_TEXTURE2D_DESC td = {0};
td.Width = width;
td.Height = height;
td.Format = DXGI_FORMAT_R16G16B16A16_FLOAT;
td.BindFlags = D3D11_BIND_RENDER_TARGET |
               D3D11_BIND_SHADER_RESOURCE;

device->CreateTexture2D(&td, NULL, &normalTex);
Write normals in pixel shader: 

float3 N = normalize(input.normal);
return float4(N, 1);
SSAO (Screen‑Space Ambient Occlusion) SSAO darkens creases and corners using depth + normals. Step 1 — Sample depth around pixel: 

float depth = depthTex.Sample(samp, uv).r;

float occlusion = 0;

for (int i = 0; i < 16; i++) {
    float2 offset = kernel[i] * radius;
    float sampleDepth = depthTex.Sample(samp, uv + offset).r;

    occlusion += sampleDepth > depth + bias ? 1 : 0;
}

occlusion /= 16;
Step 2 — Apply: 

float3 c = colorTex.Sample(samp, uv).rgb;
c *= 1 - occlusion * intensity;
return float4(c, 1);
SSR (Screen‑Space Reflections) SSR traces reflections using depth + normals. Step 1 — Compute reflection direction: 

float3 N = normalize(normalTex.Sample(samp, uv).xyz);
float3 V = normalize(cameraPos - worldPos);
float3 R = reflect(-V, N);
Step 2 — Ray march in screen space: 

float3 pos = worldPos;
float3 step = R * stepSize;

for (int i = 0; i < maxSteps; i++) {
    pos += step;

    float4 clip = mul(float4(pos,1), viewProj);
    float2 suv = clip.xy / clip.w * 0.5 + 0.5;

    float sceneDepth = depthTex.Sample(samp, suv).r;

    if (clip.z < sceneDepth + thickness) {
        return colorTex.Sample(samp, suv);
    }
}
If no hit: 

return float4(0,0,0,1);
Depth‑Based Fog Fog based on distance from camera. 

float depth = depthTex.Sample(samp, uv).r;
float fog = saturate((depth - fogStart) / (fogEnd - fogStart));

float3 c = colorTex.Sample(samp, uv).rgb;
return float4(lerp(c, fogColor, fog), 1);
Depth‑Based Edge Detection 

float d = depthTex.Sample(samp, uv).r;

float dx = abs(d - depthTex.Sample(samp, uv + float2(texel.x,0)).r);
float dy = abs(d - depthTex.Sample(samp, uv + float2(0,texel.y)).r);

float edge = saturate(dx + dy);

float3 c = colorTex.Sample(samp, uv).rgb;
return float4(c * (1-edge), 1);
Normal‑Based Effects Normal‑based outline: 

float3 N = normalTex.Sample(samp, uv).xyz;

float3 Nx = normalTex.Sample(samp, uv + float2(texel.x,0)).xyz;
float3 Ny = normalTex.Sample(samp, uv + float2(0,texel.y)).xyz;

float edge = length(N - Nx) + length(N - Ny);

float3 c = colorTex.Sample(samp, uv).rgb;
return float4(c * (1-edge), 1);
Full Screen‑Space Pipeline 

// 1. Render geometry → G-buffer (color, depth, normals)
RenderGBuffer();

// 2. SSAO → aoTex
ComputeSSAO();

// 3. SSR → reflectionTex
ComputeSSR();

// 4. Fog → fogTex
ApplyFog();

// 5. Composite all → finalTex
Composite();

// 6. Present
swap->Present(1,0);
