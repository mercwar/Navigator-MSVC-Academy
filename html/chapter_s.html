Chapter S — Maintenance

AreaRoutine
MSVC Updates 

Open Visual Studio Installer
Check for updates to Build Tools
Apply security patches monthly
MSYS2 Updates 

Run:
pacman -Syu
Close shell → reopen
pacman -Su
Schedule updates weekly
Apache + PHP Updates 

Download latest Apache build from Apache Lounge
Download latest PHP build from windows.php.net
Replace binaries in C:\Apache24 and C:\php
Re-test with info.php
GitHub Repo Maintenance 

Run:
git pull origin main
git gc --prune=now
gh issue list → resolve stale issues
WinSCP Maintenance 

Update WinSCP via built-in updater
Clear saved sessions not in use
Rotate SSH keys every 6 months
Backups 

Weekly backup routine:
xcopy C:\Projects\Navigator-MSCV D:\Backups\Navigator-MSCV /E /H /C /I
Zip backup:
7z a D:\Backups\navigator-backup.7z D:\Backups\Navigator-MSCV\*
Clean-Up 

Monthly cleanup:
del /Q C:\Projects\Navigator-MSCV\*.tmp
del /Q C:\Apache24\logs\*.old
Notes 

Navigator‑MSVC Academy maintenance ensures:
- Tools stay updated
- Logs rotated
- Backups secured
- Environment stable
