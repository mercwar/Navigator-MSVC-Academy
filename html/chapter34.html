Chapter 34 — Post‑Processing Effects: Bloom, Blur, HDR & Full‑Screen Shaders

Topic Description
What is Post‑Processing? Post‑processing applies GPU effects **after** the main scene is rendered. Common effects:
  • Bloom
  • Gaussian blur
  • HDR tonemapping
  • Color grading
  • Vignette
  • Screen‑space distortion
All post‑processing uses: Render‑to‑texture → Full‑screen quad → Pixel shader.
Render‑to‑Texture Pipeline Step 1 — Create off‑screen texture: 

D3D11_TEXTURE2D_DESC td = {0};
td.Width = width;
td.Height = height;
td.Format = DXGI_FORMAT_R8G8B8A8_UNORM;
td.BindFlags = D3D11_BIND_RENDER_TARGET |
               D3D11_BIND_SHADER_RESOURCE;

ID3D11Texture2D *sceneTex;
device->CreateTexture2D(&td, NULL, &sceneTex);
Step 2 — Create RTV + SRV: 

ID3D11RenderTargetView *sceneRTV;
device->CreateRenderTargetView(sceneTex, NULL, &sceneRTV);

ID3D11ShaderResourceView *sceneSRV;
device->CreateShaderResourceView(sceneTex, NULL, &sceneSRV);
Step 3 — Render scene into sceneRTV instead of screen.
Full‑Screen Quad Full‑screen quad vertices: 

struct FSQ {
    float2 pos;
    float2 uv;
};

FSQ quad[] = {
    { {-1,-1}, {0,1} },
    { {-1, 1}, {0,0} },
    { { 1, 1}, {1,0} },
    { { 1,-1}, {1,1} }
};
Draw with: 

context->IASetPrimitiveTopology(D3D11_PRIMITIVE_TRIANGLESTRIP);
context->Draw(4, 0);
Bloom Effect Bloom extracts bright pixels and blurs them. Step 1 — Extract bright areas: 

float4 main(float2 uv : TEXCOORD) : SV_TARGET
{
    float3 c = scene.Sample(samp, uv).rgb;
    float brightness = max(max(c.r, c.g), c.b);

    return brightness > 1.0f ? float4(c,1) : float4(0,0,0,1);
}
Step 2 — Blur horizontally Step 3 — Blur vertically Step 4 — Add back to original scene: 

finalColor = sceneColor + bloomColor * 0.7f;
Gaussian Blur Horizontal blur: 

float4 main(float2 uv : TEXCOORD) : SV_TARGET
{
    float2 texel = float2(1.0f/width, 0);

    float3 sum =
        scene.Sample(samp, uv - texel*4).rgb * 0.05 +
        scene.Sample(samp, uv - texel*3).rgb * 0.09 +
        scene.Sample(samp, uv - texel*2).rgb * 0.12 +
        scene.Sample(samp, uv - texel*1).rgb * 0.15 +
        scene.Sample(samp, uv).rgb          * 0.16 +
        scene.Sample(samp, uv + texel*1).rgb * 0.15 +
        scene.Sample(samp, uv + texel*2).rgb * 0.12 +
        scene.Sample(samp, uv + texel*3).rgb * 0.09 +
        scene.Sample(samp, uv + texel*4).rgb * 0.05;

    return float4(sum, 1);
}
Vertical blur is identical but uses: 

float2 texel = float2(0, 1.0f/height);
HDR Tonemapping HDR tonemapping compresses high brightness into displayable range. Reinhard: 

float3 hdr = scene.Sample(samp, uv).rgb;
float3 mapped = hdr / (hdr + 1.0f);
return float4(mapped, 1);
Filmic: 

float3 x = max(hdr - 0.004f, 0);
float3 mapped = (x*(6.2f*x + 0.5f)) /
                (x*(6.2f*x + 1.7f) + 0.06f);
Color Grading Apply LUT (lookup table): 

float3 c = scene.Sample(samp, uv).rgb;
float3 graded = lut.Sample(sampLUT, c).rgb;
return float4(graded, 1);
Vignette 

float2 p = uv * 2 - 1;
float v = 1 - dot(p,p);
v = saturate(v);

float3 c = scene.Sample(samp, uv).rgb;
return float4(c * v, 1);
Full Post‑Processing Pipeline 

// 1. Render scene → sceneRTV
RenderScene();

// 2. Extract bright → brightRTV
ExtractBright();

// 3. Blur → blurRTV
BlurHorizontal();
BlurVertical();

// 4. Tonemap → toneRTV
Tonemap();

// 5. Composite → backbuffer
CompositeFinal();

// 6. Present
swap->Present(1,0);
