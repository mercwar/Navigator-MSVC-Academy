Chapter O — Optimization

AreaRoutine
MSVC Compiler Flags 

Enable optimizations:
cl /O2 /GL /DNDEBUG source.cpp
Link with:
link /LTCG /OPT:REF /OPT:ICF
MSYS2 Build Speed 

Use Ninja for parallel builds:
cmake -G "Ninja" ..
ninja -j8
Apache Performance 

Edit httpd.conf:
Enable KeepAlive On
Set MaxKeepAliveRequests 100
Enable mod_deflate for compression
Enable mod_cache for static assets
PHP Settings 

Edit php.ini:
opcache.enable=1
opcache.memory_consumption=128
opcache.max_accelerated_files=4000
display_errors=Off
log_errors=On
WinSCP Transfer Speed 

Preferences → Transfer:
Enable parallel transfers
Set max connections = 4
Enable compression for SFTP
7-Zip Compression 

High compression:
7z a -t7z -mx9 build.7z project\*
Balanced speed:
7z a -tzip -mx5 build.zip project\*
