Chapter 25 — Windows Services & Service Control Manager (SCM)

Topic Description
What is a Windows Service? A Windows Service is a background application managed by the Service Control Manager (SCM). Services can:
  • Start automatically at boot
  • Run without user login
  • Restart on failure
  • Run under specific accounts
  • Respond to control codes (stop, pause, resume)
Services use a special entry point: ServiceMain.
Service Control Manager (SCM) SCM manages all services. Open SCM: 

SC_HANDLE scm = OpenSCManager(
    NULL,
    NULL,
    SC_MANAGER_ALL_ACCESS
);
Installing a Service 

SC_HANDLE svc = CreateService(
    scm,
    "MyService",
    "My Demo Service",
    SERVICE_ALL_ACCESS,
    SERVICE_WIN32_OWN_PROCESS,
    SERVICE_AUTO_START,
    SERVICE_ERROR_NORMAL,
    "C:\\MyService.exe",
    NULL, NULL, NULL,
    NULL, NULL
);
Start automatically at boot with SERVICE_AUTO_START.
Starting a Service 

StartService(svc, 0, NULL);
Deleting a Service 

DeleteService(svc);
Service Entry Point Services begin execution in ServiceMain: 

void WINAPI ServiceMain(DWORD argc, LPSTR *argv)
{
    // register service control handler
}
The service must register a handler: 

SERVICE_STATUS_HANDLE hStatus =
    RegisterServiceCtrlHandler("MyService", ServiceCtrl);
Service Control Handler 

void WINAPI ServiceCtrl(DWORD ctrl)
{
    switch (ctrl) {
        case SERVICE_CONTROL_STOP:
            // stop service
            break;

        case SERVICE_CONTROL_PAUSE:
            // pause service
            break;

        case SERVICE_CONTROL_CONTINUE:
            // resume service
            break;
    }
}
SCM sends control codes to the service.
Updating Service Status 

SERVICE_STATUS status = {0};
status.dwServiceType = SERVICE_WIN32_OWN_PROCESS;
status.dwCurrentState = SERVICE_RUNNING;

SetServiceStatus(hStatus, &status);
Service Dispatch Table 

SERVICE_TABLE_ENTRY table[] = {
    { "MyService", ServiceMain },
    { NULL, NULL }
};

StartServiceCtrlDispatcher(table);
This function blocks until the service stops.
Full Minimal Service Example 

#include <windows.h>

SERVICE_STATUS_HANDLE hStatus;
SERVICE_STATUS status;

void WINAPI ServiceCtrl(DWORD ctrl)
{
    switch (ctrl) {
        case SERVICE_CONTROL_STOP:
            status.dwCurrentState = SERVICE_STOPPED;
            SetServiceStatus(hStatus, &status);
            break;
    }
}

void WINAPI ServiceMain(DWORD argc, LPSTR *argv)
{
    hStatus = RegisterServiceCtrlHandler("DemoService", ServiceCtrl);

    status.dwServiceType = SERVICE_WIN32_OWN_PROCESS;
    status.dwCurrentState = SERVICE_RUNNING;
    status.dwControlsAccepted = SERVICE_ACCEPT_STOP;

    SetServiceStatus(hStatus, &status);

    // service loop
    while (status.dwCurrentState == SERVICE_RUNNING) {
        Sleep(1000);
    }
}

int main(void)
{
    SERVICE_TABLE_ENTRY table[] = {
        { "DemoService", ServiceMain },
        { NULL, NULL }
    };

    StartServiceCtrlDispatcher(table);
    return 0;
}
Service Accounts Services can run under:
  • LocalSystem
  • LocalService
  • NetworkService
  • Custom user accounts
Example: 

CreateService(
    scm,
    "MyService",
    "My Demo Service",
    SERVICE_ALL_ACCESS,
    SERVICE_WIN32_OWN_PROCESS,
    SERVICE_AUTO_START,
    SERVICE_ERROR_NORMAL,
    "C:\\MyService.exe",
    NULL, NULL, NULL,
    ".\\MyUser",
    "Password123"
);
Service Failure Actions Configure automatic restart: 

SC_ACTION actions[1];
actions[0].Type = SC_ACTION_RESTART;
actions[0].Delay = 1000;

SERVICE_FAILURE_ACTIONS sfa = {0};
sfa.dwResetPeriod = 0;
sfa.cActions = 1;
sfa.lpsaActions = actions;

ChangeServiceConfig2(svc, SERVICE_CONFIG_FAILURE_ACTIONS, &sfa);
