Chapter 20 — WinSock Networking (TCP, UDP & Async Communication)

Topic Description
What is WinSock? WinSock is the Windows networking API used for:
  • TCP connections
  • UDP datagrams
  • DNS lookups
  • Async network events
  • Client/server communication
WinSock is based on Berkeley sockets but uses Windows‑specific initialization and cleanup.
Initializing WinSock Every WinSock program begins with WSAStartup: 

WSADATA wsa;
WSAStartup(MAKEWORD(2,2), &wsa);
Cleanup: 

WSACleanup();
Creating a Socket 

SOCKET s = socket(AF_INET, SOCK_STREAM, IPPROTO_TCP);
Parameters:
  • AF_INET — IPv4
  • SOCK_STREAM — TCP
  • IPPROTO_TCP — TCP protocol
TCP Client 

SOCKET s = socket(AF_INET, SOCK_STREAM, IPPROTO_TCP);

struct sockaddr_in server;
server.sin_family = AF_INET;
server.sin_port   = htons(8080);
server.sin_addr.s_addr = inet_addr("127.0.0.1");

connect(s, (struct sockaddr*)&server, sizeof(server));

send(s, "Hello", 5, 0);

char buffer[256];
int r = recv(s, buffer, 256, 0);

closesocket(s);
TCP is connection‑oriented and reliable.
TCP Server 

SOCKET s = socket(AF_INET, SOCK_STREAM, IPPROTO_TCP);

struct sockaddr_in addr;
addr.sin_family = AF_INET;
addr.sin_port   = htons(8080);
addr.sin_addr.s_addr = INADDR_ANY;

bind(s, (struct sockaddr*)&addr, sizeof(addr));
listen(s, 5);

SOCKET client = accept(s, NULL, NULL);

send(client, "Welcome!", 8, 0);

closesocket(client);
closesocket(s);
Servers use bind → listen → accept.
UDP Socket 

SOCKET s = socket(AF_INET, SOCK_DGRAM, IPPROTO_UDP);
UDP send: 

sendto(s, "Ping", 4, 0,
       (struct sockaddr*)&addr, sizeof(addr));
UDP receive: 

recvfrom(s, buffer, 256, 0,
         (struct sockaddr*)&from, &fromLen);
UDP is connectionless and fast.
Async Networking (WSAAsyncSelect) WinSock can send network events to a window: 

WSAAsyncSelect(s, hwnd, WM_USER + 100,
               FD_READ | FD_WRITE | FD_CONNECT | FD_CLOSE);
Handle in WndProc: 

case WM_USER + 100:
    if (WSAGETSELECTEVENT(lParam) == FD_READ) {
        recv(s, buffer, 256, 0);
    }
    return 0;
This allows non‑blocking networking in GUI apps.
Non‑Blocking Sockets 

u_long mode = 1;
ioctlsocket(s, FIONBIO, &mode);
Now recv() returns immediately instead of blocking.
DNS Lookup (getaddrinfo) 

ADDRINFO hints = {0};
hints.ai_family = AF_INET;

ADDRINFO *result;
getaddrinfo("example.com", "80", &hints, &result);
Extract IP: 

struct sockaddr_in *addr =
    (struct sockaddr_in*)result->ai_addr;

char *ip = inet_ntoa(addr->sin_addr);
Full TCP Client Example 

#include <winsock2.h>
#include <ws2tcpip.h>
#pragma comment(lib, "ws2_32.lib")

int main(void)
{
    WSADATA wsa;
    WSAStartup(MAKEWORD(2,2), &wsa);

    SOCKET s = socket(AF_INET, SOCK_STREAM, IPPROTO_TCP);

    struct sockaddr_in server;
    server.sin_family = AF_INET;
    server.sin_port   = htons(8080);
    server.sin_addr.s_addr = inet_addr("127.0.0.1");

    connect(s, (struct sockaddr*)&server, sizeof(server));

    send(s, "Hello Server", 12, 0);

    char buffer[256];
    int r = recv(s, buffer, 256, 0);

    closesocket(s);
    WSACleanup();
    return 0;
}
