Chapter D — Apache HTTP Server Installation

StepRoutine
Download 

Go to: https://www.apachelounge.com/download/
Download latest Win64 VC build (e.g., httpd-2.4.59-win64-VS17.zip)
Install 

Extract ZIP to: C:\Apache24
Ensure folder structure: C:\Apache24\bin, C:\Apache24\conf, C:\Apache24\htdocs
Configure 

Edit C:\Apache24\conf\httpd.conf
Set:
ServerRoot "C:/Apache24"
Listen 80
DocumentRoot "C:/Apache24/htdocs"

  Options Indexes FollowSymLinks
  AllowOverride All
  Require all granted
Verify 

Open CMD → cd C:\Apache24\bin
Run: httpd -t
Output: Syntax OK
Service 

Install service: httpd.exe -k install
Start service: httpd.exe -k start
Stop service: httpd.exe -k stop
Restart service: httpd.exe -k restart
Test 

Open browser → http://localhost/
Default page should display "It works!"
Notes 

Navigator‑MSVC Academy uses Apache as the primary web server.
Logs: C:\Apache24\logs\error.log and access.log
Enable mod_ssl for HTTPS if required.
