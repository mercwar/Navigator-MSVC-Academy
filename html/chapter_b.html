Chapter B — MSYS2 / UCRT64 Installation

StepRoutine
Download 

Go to: https://www.msys2.org/
Download latest installer (e.g., msys2-x86_64-20240504.exe)
Install to: C:\msys64
Update Core 

Open MSYS2 MSYS shell
Run: pacman -Syu
Close → reopen shell
Run: pacman -Su
Install UCRT64 Toolchain 

Open UCRT64 shell (C:\msys64\ucrt64.exe)
Run:
pacman -S --needed ucrt64/mingw-w64-ucrt-x86_64-gcc
pacman -S --needed ucrt64/mingw-w64-ucrt-x86_64-clang
pacman -S --needed ucrt64/mingw-w64-ucrt-x86_64-cmake
pacman -S --needed ucrt64/mingw-w64-ucrt-x86_64-ninja
Verify 

Run:
gcc --version
clang --version
cmake --version
ninja --version
Environment 

Add to PATH (CMD only):
set PATH=C:\msys64\ucrt64\bin;%PATH%
Notes 

Navigator‑MSVC Academy uses UCRT64 for:
- GF engine builds
- Memory block tools
- LLM command execution
CMD only — no PowerShell
