Chapter 1 — Introduction to C

Topic Description
What is C? C is a general-purpose, compiled programming language used for operating systems, embedded systems, game engines, compilers, firmware, and high-performance applications. It provides low-level memory control while remaining readable and structured.
Why C matters C is the foundation of modern computing. Most languages (C++, Rust, Go, Python interpreters) are built on top of C. Understanding C gives you direct insight into how software interacts with hardware and operating systems.
Compilation model C is a compiled language. Source code (.c) is transformed into machine code (.exe, .out) by a compiler such as GCC or MSVC. This produces fast, efficient binaries.
Basic structure of a C program 

#include <stdio.h>

int main(void) {
    printf("Hello, C world!\\n");
    return 0;
}
  • #include <stdio.h> — imports standard I/O functions.
  • int main(void) — program entry point.
  • printf() — prints text to the console.
  • return 0; — signals successful execution.
Key features of C
  • Manual memory control (malloc, free)
  • Low-level pointer access
  • Fast execution
  • Portable across platforms
  • Small runtime footprint
Common uses
  • Operating systems (Windows, Linux kernel)
  • Embedded systems (microcontrollers, IoT)
  • Game engines (idTech, early Unreal)
  • Compilers and interpreters
  • High-performance libraries
Development tools
  • GCC (Linux/macOS)
  • Clang (cross-platform)
  • MSVC (Windows)
  • Make / CMake for build automation
  • VSCode / Visual Studio / Vim / Emacs
Compiling your first program Linux/macOS (GCC) 

gcc main.c -o main
./main
Windows (MSVC) 

cl main.c
main.exe
