Chapter 21 — File I/O & Memory‑Mapped Files in WinAPI

Topic Description
WinAPI File Handles WinAPI uses HANDLE for files instead of C’s FILE*. 

HANDLE hFile = CreateFile(
    "data.txt",
    GENERIC_READ | GENERIC_WRITE,
    0,
    NULL,
    OPEN_ALWAYS,
    FILE_ATTRIBUTE_NORMAL,
    NULL
);
Flags:
  • GENERIC_READ — read access
  • GENERIC_WRITE — write access
  • OPEN_ALWAYS — create if missing
  • FILE_ATTRIBUTE_NORMAL — standard file
Reading from a File 

char buffer[256];
DWORD bytesRead;

ReadFile(hFile, buffer, sizeof(buffer), &bytesRead, NULL);
bytesRead contains the number of bytes returned.
Writing to a File 

char text[] = "Hello WinAPI!";
DWORD written;

WriteFile(hFile, text, strlen(text), &written, NULL);
Closing a File 

CloseHandle(hFile);
Always close file handles to avoid leaks.
Moving the File Pointer 

SetFilePointer(hFile, 0, NULL, FILE_BEGIN);
Options:
  • FILE_BEGIN
  • FILE_CURRENT
  • FILE_END
Getting File Size 

DWORD size = GetFileSize(hFile, NULL);
Memory‑Mapped Files Memory mapping allows you to treat a file as RAM. Step 1 — Create file mapping: 

HANDLE hMap = CreateFileMapping(
    hFile,
    NULL,
    PAGE_READWRITE,
    0,
    0,
    NULL
);
Step 2 — Map view: 

char *data = (char*)MapViewOfFile(
    hMap,
    FILE_MAP_ALL_ACCESS,
    0, 0,
    0
);
Now data points directly into the file. Step 3 — Modify: 

strcpy(data, "Memory mapped write!");
Step 4 — Cleanup: 

UnmapViewOfFile(data);
CloseHandle(hMap);
File Attributes 

DWORD attr = GetFileAttributes("data.txt");
Common attributes:
  • FILE_ATTRIBUTE_READONLY
  • FILE_ATTRIBUTE_HIDDEN
  • FILE_ATTRIBUTE_SYSTEM
  • FILE_ATTRIBUTE_DIRECTORY
Creating a Directory 

CreateDirectory("myfolder", NULL);
Deleting a Directory 

RemoveDirectory("myfolder");
Finding Files (Search) 

WIN32_FIND_DATA fd;
HANDLE hFind = FindFirstFile("*.txt", &fd);

do {
    printf("%s\n", fd.cFileName);
} while (FindNextFile(hFind, &fd));

FindClose(hFind);
Copying & Moving Files Copy: 

CopyFile("a.txt", "b.txt", FALSE);
Move: 

MoveFile("a.txt", "folder\\a.txt");
Deleting Files 

DeleteFile("old.txt");
Full File I/O + Memory Map Example 

#include <windows.h>
#include <stdio.h>

int main(void)
{
    HANDLE hFile = CreateFile(
        "demo.txt",
        GENERIC_READ | GENERIC_WRITE,
        0,
        NULL,
        OPEN_ALWAYS,
        FILE_ATTRIBUTE_NORMAL,
        NULL
    );

    HANDLE hMap = CreateFileMapping(
        hFile, NULL,
        PAGE_READWRITE,
        0, 0, NULL
    );

    char *data = (char*)MapViewOfFile(
        hMap,
        FILE_MAP_ALL_ACCESS,
        0, 0, 0
    );

    strcpy(data, "Hello from memory mapping!");

    UnmapViewOfFile(data);
    CloseHandle(hMap);
    CloseHandle(hFile);

    return 0;
}
