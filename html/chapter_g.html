Chapter G — 7‑Zip Installation

StepRoutine
Download 

Go to: https://www.7-zip.org/
Download latest 64-bit installer (e.g., 7z2400-x64.exe)
Install 

Run installer → Accept license
Install to: C:\Program Files\7-Zip
Verify 

Open CMD
Run: 7z
Output should display 7-Zip version info
Basic Usage 

Create ZIP:
7z a archive.zip files\*

Extract ZIP:
7z x archive.zip -ooutput

List contents:
7z l archive.zip
Batch Integration 

Example: ZIP-BUILD.bat
@echo off
setlocal
set SRC=C:\Project\Demo
set OUT=C:\Project\build.zip
"C:\Program Files\7-Zip\7z.exe" a -tzip "%OUT%" "%SRC%\*"
endlocal
Notes 

Navigator‑MSVC Academy uses 7‑Zip for:
- Packaging builds
- Multi-ZIP routines
- Memory block compression
CMD only — no PowerShell
