Chapter 36 — Physics: Rigid Bodies, Collisions, Forces & Simulation Loops

Topic Description
Physics Engine Overview A physics engine simulates motion using:
  • Rigid bodies
  • Forces (gravity, drag, impulses)
  • Collision detection
  • Collision response
  • Integration (Euler, RK4)
Real‑time physics runs every frame (60–240 FPS).
Rigid Body Structure 

struct RigidBody {
    XMFLOAT3 pos;
    XMFLOAT3 vel;
    XMFLOAT3 acc;
    float mass;
    bool isStatic;
};
Update rule: 

vel += acc * dt;
pos += vel * dt;
acc = {0,0,0};
Applying Forces 

void ApplyForce(RigidBody &rb, XMFLOAT3 f)
{
    if (!rb.isStatic)
        rb.acc.x += f.x / rb.mass,
        rb.acc.y += f.y / rb.mass,
        rb.acc.z += f.z / rb.mass;
}
Gravity: 

ApplyForce(rb, {0, -9.8f * rb.mass, 0});
Drag: 

rb.vel *= 0.99f;
Collision Detection (AABB) Axis‑aligned bounding box: 

struct AABB {
    XMFLOAT3 min;
    XMFLOAT3 max;
};

bool Intersect(const AABB &a, const AABB &b)
{
    return (a.min.x <= b.max.x && a.max.x >= b.min.x) &&
           (a.min.y <= b.max.y && a.max.y >= b.min.y) &&
           (a.min.z <= b.max.z && a.max.z >= b.min.z);
}
Collision Response Simple bounce: 

rb.vel = rb.vel * -0.5f;
Position correction: 

rb.pos.y = groundY;
Impulse‑based: 

float j = -(1 + restitution) * dot(vRel, normal)
          / (1/rbA.mass + 1/rbB.mass);

rbA.vel += normal * ( j / rbA.mass );
rbB.vel -= normal * ( j / rbB.mass );
Integration Methods Euler: 

vel += acc * dt;
pos += vel * dt;
Semi‑implicit Euler: 

vel += acc * dt;
pos += vel * dt;

Real‑Time Simulation Loop 

while (running) {
    float dt = GetDeltaTime();

    // 1. Apply forces
    for (auto &rb : bodies)
        ApplyForce(rb, gravity);

    // 2. Integrate
    for (auto &rb : bodies)
        Integrate(rb, dt);

    // 3. Collisions
    for (auto &a : bodies)
        for (auto &b : bodies)
            if (&a != &b && Intersect(a.box, b.box))
                ResolveCollision(a, b);

    // 4. Render
    RenderScene();
}
Sphere Collision 

bool SphereHit(XMFLOAT3 a, float ra,
               XMFLOAT3 b, float rb)
{
    float dx = a.x - b.x;
    float dy = a.y - b.y;
    float dz = a.z - b.z;
    float dist2 = dx*dx + dy*dy + dz*dz;
    float r = ra + rb;
    return dist2 <= r*r;
}
