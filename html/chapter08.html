Chapter 8 — Functions in C

Concept Description
What is a function? A function is a reusable block of code that performs a specific task. Functions help organize programs, reduce repetition, and improve readability.

Every function has:
  • a return type
  • a name
  • parameters (optional)
  • a body
Basic function example 

int add(int a, int b) {
    return a + b;
}
  • int — return type
  • add — function name
  • (int a, int b) — parameters
  • return a + b; — returned value
Calling a function 

int result = add(3, 4);
printf("%d\\n", result);
The function executes and returns a value.
Functions with no parameters 

void greet(void) {
    printf("Hello!\\n");
}
void means no return value.
Functions returning void 

void printLine(void) {
    printf("----------------\\n");
}
Used for actions rather than calculations.
Functions with multiple parameters 

void showInfo(char name[], int age) {
    printf("%s is %d years old\\n", name, age);
}
Function prototypes A prototype tells the compiler a function exists before it is used. 

int add(int a, int b);  // prototype

int main(void) {
    printf("%d\\n", add(5, 7));
}
Prototypes are usually placed at the top of the file or in header files.
Header files Functions can be declared in .h files: 

// mathutils.h
int add(int a, int b);
And implemented in .c files: 

// mathutils.c
int add(int a, int b) {
    return a + b;
}
This is the foundation of modular C programming.
Static functions 

static int internalCalc(int x) {
    return x * 2;
}
static limits visibility to the current file.
Recursion A function calling itself: 

int factorial(int n) {
    if (n == 0) return 1;
    return n * factorial(n - 1);
}
Powerful but must be used carefully.
Inline functions (C99) 

inline int square(int x) {
    return x * x;
}
Suggests the compiler replace the call with the function body for speed.
Passing by value C always passes arguments by value: 

void change(int x) {
    x = 100;
}
The original variable is not modified.
Passing by reference (using pointers) 

void change(int *x) {
    *x = 100;
}
Now the original variable is modified.
Example: Full function program 

#include <stdio.h>

int add(int a, int b) {
    return a + b;
}

void show(int value) {
    printf("Value: %d\\n", value);
}

int main(void) {
    int result = add(10, 20);
    show(result);
    return 0;
}
