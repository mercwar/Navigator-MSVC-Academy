Chapter F — AVIS Vector Spine & DXGI Root

TopicDescription
AVIS Vector Spine AVIS maintains the **spine**, a unified pointer map: 

struct AVIS_SPINE {
    CBORD   *cbord;
    CYBORG  *cyborg;
    CYHY    *cyhy;
    MERCG   *mercg;
    DXVector *dx;
};
This spine is the authoritative routing table for all DirectX calls.
DXGI Root CBORD initializes DXGI and registers it into AVIS: 

IDXGIFactory *factory;
CreateDXGIFactory(__uuidof(IDXGIFactory),(void**)&factory);
avis.spine.dx->factory = factory;
DXGI is the **root** of all DirectX vectoring.
Subsystem Binding AVIS binds subsystems: 

avis.spine.cbord  = &cbord;
avis.spine.cyborg = &cyborg;
avis.spine.cyhy   = &cyhy;
avis.spine.mercg  = &mercg;
DXVector Exposure DXVector is exposed upward: 

avis.spine.dx = &cbord.dx;
All subsystems vector through this single DX root.
