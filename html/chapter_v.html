Memory Blocks 

Create file: memory_block.txt
Content:
[BLOCK]
Academy=Navigator-MSCV
Engine=FireGem
Pipeline=DX
[/BLOCK]

Batch loader:
for /f "tokens=1,2 delims==" %%a in (memory_block.txt) do (
  set %%a=%%b
)
FireGem Hooks 

Hook file: firegem_hook.bat
@echo off
echo FireGem LLM hook triggered
call llm_exec.bat "cl /O2 main.cpp -Fe:firegem.exe"
