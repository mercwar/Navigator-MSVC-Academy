Chapter 17 — Dialogs & Message Boxes in WinAPI

Component Description
Message Boxes The simplest dialog in WinAPI. 

MessageBox(hwnd, "Operation complete.", "Info", MB_OK);
Common styles:
  • MB_OK — OK button
  • MB_YESNO — Yes / No
  • MB_ICONERROR — error icon
  • MB_ICONWARNING — warning icon
  • MB_ICONINFORMATION — info icon
Example: 

int r = MessageBox(hwnd, "Delete file?", "Confirm", MB_YESNO | MB_ICONWARNING);
if (r == IDYES) { /* delete */ }
What is a Dialog? A dialog is a special window used for:
  • Settings
  • Forms
  • Input boxes
  • File operations
  • About boxes
Dialogs can be:
  • Modal — blocks the main window
  • Modeless — acts like a normal window
Modal Dialog Modal dialogs use DialogBox: 

DialogBox(hInstance, MAKEINTRESOURCE(IDD_MYDIALOG),
          hwnd, DialogProc);
They block until closed.
Modeless Dialog Modeless dialogs use CreateDialog: 

HWND dlg = CreateDialog(hInstance,
                        MAKEINTRESOURCE(IDD_MYDIALOG),
                        hwnd, DialogProc);

ShowWindow(dlg, SW_SHOW);
They behave like normal windows and do not block.
Dialog Resource (.rc file) Dialogs are defined in resource scripts: 

IDD_MYDIALOG DIALOGEX 0, 0, 200, 120
STYLE DS_MODALFRAME | WS_POPUP | WS_CAPTION | WS_SYSMENU
CAPTION "Settings"
FONT 8, "MS Shell Dlg"
BEGIN
    LTEXT "Name:", -1, 10, 10, 40, 12
    EDITTEXT IDC_NAME, 60, 10, 120, 12
    DEFPUSHBUTTON "OK", IDOK, 40, 80, 50, 14
    PUSHBUTTON "Cancel", IDCANCEL, 110, 80, 50, 14
END
This creates:
  • Static text
  • Edit box
  • OK button
  • Cancel button
Dialog Procedure Dialogs use DialogProc instead of WndProc: 

INT_PTR CALLBACK DialogProc(HWND dlg, UINT msg,
                            WPARAM wParam, LPARAM lParam)
{
    switch (msg) {

        case WM_INITDIALOG:
            return TRUE;

        case WM_COMMAND:
            switch (LOWORD(wParam)) {
                case IDOK:
                    EndDialog(dlg, IDOK);
                    return TRUE;

                case IDCANCEL:
                    EndDialog(dlg, IDCANCEL);
                    return TRUE;
            }
            break;
    }
    return FALSE;
}
EndDialog closes modal dialogs.
Getting Input from Dialog Controls 

char buffer[256];
GetDlgItemText(dlg, IDC_NAME, buffer, 256);
Setting text: 

SetDlgItemText(dlg, IDC_NAME, "Default");
Full Dialog Example 

#include <windows.h>
#include "resource.h"

INT_PTR CALLBACK DialogProc(HWND dlg, UINT msg,
                            WPARAM wParam, LPARAM lParam)
{
    switch (msg) {

        case WM_INITDIALOG:
            SetDlgItemText(dlg, IDC_NAME, "Demon");
            return TRUE;

        case WM_COMMAND:
            switch (LOWORD(wParam)) {
                case IDOK: {
                    char name[256];
                    GetDlgItemText(dlg, IDC_NAME, name, 256);
                    MessageBox(dlg, name, "You Entered", MB_OK);
                    EndDialog(dlg, IDOK);
                    return TRUE;
                }

                case IDCANCEL:
                    EndDialog(dlg, IDCANCEL);
                    return TRUE;
            }
            break;
    }
    return FALSE;
}

int WINAPI WinMain(HINSTANCE hInstance,
                   HINSTANCE hPrevInstance,
                   LPSTR lpCmdLine,
                   int nCmdShow)
{
    DialogBox(hInstance, MAKEINTRESOURCE(IDD_MYDIALOG),
              NULL, DialogProc);
    return 0;
}
